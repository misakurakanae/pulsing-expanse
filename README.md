# 📰 あなた専用ニュース - AIパーソナライズドニュースアプリ

AIがあなたの興味に合わせて最新ニュースを要約・配信するWebアプリです。

## ✨ 主な機能

- 🤖 **Gemini AI要約**: 長いニュース記事を3-4文で簡潔に要約
- ⭐ **5段階評価**: ニュースに星1-5の評価をつけて学習
- 🚫 **興味なしボタン**: 興味のないニュースを除外
- 🎯 **キーワード登録**: 好みのトピックを登録して自動フィルタリング
- 📊 **8つのニュースソース**:
  - 東洋経済オンライン
  - ダイヤモンド・オンライン
  - プレジデントオンライン
  - 現代ビジネス
  - ITmedia NEWS
  - Impress Watch
  - GIGAZINE
  - 佐賀新聞

## 🚀 セットアップ手順

### 1. Node.jsのインストール

まずNode.jsをインストールしてください（推奨: v18以上）:
https://nodejs.org/

### 2. 依存関係のインストール

```bash
npm install
```

### 3. Gemini APIキーの取得

1. [Google AI Studio](https://makersuite.google.com/app/apikey) にアクセス
2. 「Create API Key」をクリック
3. APIキーをコピー

### 4. 環境変数の設定

`.env.example`をコピーして`.env.local`を作成:

```bash
copy .env.example .env.local
```

`.env.local`を編集してAPIキーを設定:

```env
GEMINI_API_KEY=あなたのAPIキーをここに貼り付け
```

### 5. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで http://localhost:3000 を開いてください！

## 📖 使い方

### キーワード登録
1. 「興味キーワード」入力欄にキーワードを入力（例: AI, スタートアップ, 経済）
2. 「追加」ボタンをクリック
3. キーワードの×ボタンで削除可能

### ニュースフィルタリング
- **キーワードフィルター**: 登録したキーワードを含むニュースのみ表示
- **AI要約を使用**: Gemini AIでニュースを自動要約（少し時間がかかります）

### ニュース評価
- ⭐ **1-5星**: ニュースの評価をつける（5が最高）
- 🚫 **興味なし**: このニュースに興味がない場合にクリック

> 評価はブラウザのローカルストレージに保存されます（現バージョンでは個人利用のみ）

## 💡 トラブルシューティング

### ニュースが表示されない
- RSSフィードがアクセス不可の可能性があります
- 数分待ってから「更新」ボタンをクリック

### AI要約が生成されない
- `.env.local`のGemini APIキーが正しいか確認
- Gemini APIの無料枠制限（60リクエスト/分）に達している可能性

### Node.jsコマンドが認識されない
- Node.jsが正しくインストールされているか確認
- コマンドプロンプトを再起動
